# 🔀 n8n Workflow Visualizer

**Analysiere und visualisiere deine n8n-Workflows** — kostenlos und sofort im Browser.

Dieses Tool von [Noyra-X](https://noyra-x.de) hilft Automatisierern dabei, n8n-Workflows auf einen Blick zu verstehen: Welche Nodes sind verbunden? Wie komplex ist der Workflow? Wo gibt es Optimierungspotenzial?

## Wie es funktioniert

1. **Workflow exportieren** — In n8n: Workflow öffnen → Menü → Download
2. **JSON hochladen** — Datei in das Upload-Feld ziehen
3. **Analysieren** — Button klicken und Ergebnisse erhalten
